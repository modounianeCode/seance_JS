Developpement web front-end

Le Developpement web est un domaine tres vaste ou on a plusieurs choses à apprendre et plusieurs specialités dont le full-stack, le back-end et le front-end.
Ce dernier, qui est le theme d'aujourd'hui nous permet de faire beaucoup de choses notamment la responsivite, le style, le dynamisme d'un site ou d'une application.
Les principaux langages de ce domaine que nous allons aborder sont le HTML, le CSS et le JS

HTML

Le le HTML est le langage de balisage hyper text qui nous permet de
de structurer le contenu de note page web en utilisant des balises paires et des balises orphelines

CSS

le CSS est le langage qui permet de styliser la page.
En d'autres termes, il permet d'ajoutrer des couleurs, des marges, des bordures etc. Il utilise des selecteurs (balise, class, id) , des proprietes(color, background-color, padding, margin, ...) et des valeurs(red, green, 180px, ... ).

JS

le JS est le langage qui rend nos pages web interactifs en utilisant des fonctions et plusieurs evenements notamment onclick, onsubmit, etc.
